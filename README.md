# Bot TG Project

This repository contains the source code and scripts for the Bot TG project.

## Files

- `create_montage.py`: Python script to create montages from slide images.
- `pptx_to_img.py`: Script to rasterize PowerPoint slides into images for edge overflow checks.
- `answer.js`: Main script containing the bot logic.
- `slides_template.js`: Template for creating slides presentations.
- `placeholder_light_gray_block.png`: Placeholder image used when content is missing on a slide.
- `79fba4a9-0919-46b7-ba79-7d1c2a6c67bc.png`: Example image used in slides.
- `package.json` & `package-lock.json`: Node project configuration files.

## Usage

Run `npm install` to install dependencies, then run the scripts as needed.
